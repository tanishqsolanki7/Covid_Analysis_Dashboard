# Covid Analysis Dashboard 

# Overview
An interactive Tableau dashboard analyzing and visualizing 
COVID-19 data across Indian states — tracking confirmed cases, 
recoveries, and deaths over time.

# Visuals Included
- Map Chart : Confirmed Cases State-wise across India
- Bar Chart : Top 10 States by Covid Cases
- Line Chart : Weekly Analysis of Covid Deaths over time

# Key Insights
- Total Confirmed Cases : 5,45,16,78,687
- Total Deaths : 7,33,89,005
- Total Cured : 5,04,61,25,452
- Maharashtra recorded the highest cases at 1,121,491K
- Significant death spike observed between Dec 2020 - Jun 2021

# Tools Used
- Tableau Desktop
- Data Visualization
- Map Visualization (Mapbox)
